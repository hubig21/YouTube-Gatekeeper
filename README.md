# YouTube 頻道過濾器 — 使用說明 v1.0.4

強大的多層式 YouTube 影片過濾腳本，可依頻道、關鍵字、時長、語言、直播與會員專屬內容過濾影片。

---

## 🚀 安裝方式

1. 安裝 [Tampermonkey](https://www.tampermonkey.net/)（支援 Firefox / Chrome / Edge）
2. 點擊腳本安裝連結，在 Tampermonkey 頁面確認安裝
3. 重新整理 YouTube 頁面，頂欄右側出現 🔍 圖示即表示安裝成功

---

## 🎛️ 基本操作

| 操作 | 說明 |
|------|------|
| 🔍 **左鍵** | 開啟 / 關閉管理面板 |
| 🔍 **右鍵** | 快速切換過濾器（啟用 / 暫停） |
| Tampermonkey 選單 | 也可從擴充圖示選單開啟面板或切換過濾器 |
| **滑鼠移到卡片上** | 卡片底部滑出「🚫 隱藏此頻道 ｜ ⭐ 加入白名單」快捷按鈕 |

---

## 🚫 隱藏名單（Blocklist）

在「🚫 隱藏名單」Tab 中新增要隱藏的頻道：

- 輸入**頻道名稱**（如：`某某頻道`）或 **@handle**（如：`@channelhandle`）
- 該頻道的所有影片將從首頁、搜尋、側邊推薦全面消失
- 可使用**標籤**分類管理大量隱藏頻道
- 新增時可勾選「**⌥R.* Regex 模式**」使用正規表達式批次比對頻道名稱（詳見下方 Regex 說明）

---

## ⭐ 白名單（Whitelist）

白名單頻道**永遠顯示**，不受任何過濾規則影響：

- 優先級最高，即使頻道同時在隱藏名單中，白名單仍勝出
- 適合在啟用大範圍關鍵字過濾或語言過濾時，保護特定頻道不被誤殺
- 同樣支援 **⌥R.* Regex 模式**

---

## 🔤 關鍵字過濾

建立**關鍵字群組**，依標題或描述過濾影片：

1. 點「＋ 新增群組」→ 輸入群組名稱（如：`業配廣告`）
2. 設定套用範圍：**標題**、**描述**或兩者
3. 在群組中新增關鍵字，支援**一般文字**或 **Regex 正規表達式**
4. 群組可整組開啟 / 關閉，方便臨時停用

---

## ⚙️ 其他過濾設定

| 設定項目 | 說明 |
|---------|------|
| ⏱ **影片時長** | 設定最短 / 最長秒數，超出範圍的影片被隱藏（0 = 不限） |
| 📡 **直播 / 首播** | 隱藏進行中直播 / 隱藏即將首播的影片 |
| 👑 **會員專屬** | 隱藏需要付費才能觀看的會員限定影片 |
| 🌐 **語言過濾** | 依標題字元自動偵測語言，可「過濾指定語言」或「只顯示指定語言」 |

語言過濾支援 20+ 種語言，也可手動輸入 BCP-47 語言代碼（如 `th`、`vi`、`id`）。

設定完成後務必點「**💾 儲存設定**」才會生效。

## ⌥R.* Regex 模式 — 詳細說明

Regex（正規表達式）讓你用一條規則比對大量頻道名稱或關鍵字。

### 格式規則

- **直接輸入 pattern**，不需加 `/` 開頭或 `/i` 結尾
- 預設**不分大小寫**（相當於 `/i` flag 永遠開啟）
- 不支援旗標語法，如輸入 `memes/i` 會將 `/i` 視為 pattern 的一部分

### 頻道名稱 Regex（隱藏名單 / 白名單）

勾選「⌥R.* Regex 模式」後，輸入 pattern 對頻道名稱進行比對：

| 範例 Pattern | 效果 |
|-------------|------|
| `memes` | 頻道名稱中**任意位置**含有 `memes`（不分大小寫） |
| `^memes` | 頻道名稱**開頭**是 `memes` |
| `memes$` | 頻道名稱**結尾**是 `memes` |
| `\bmemes\b` | 頻道名稱中 `memes` 是完整獨立的詞 |
| `(shorts\|clip)` | 比對含有 `shorts` 或 `clip` 的頻道名稱 |
| `^NASA$` | 精確完整比對（名稱就是 `NASA`，大小寫不限） |

> **注意**：Regex 頻道過濾只比對**頻道名稱**，不比對頻道 ID（@handle）。若要用 ID 過濾，請輸入精確的 @handle 並關閉 Regex 模式。

### ✅ 何時適合用 Regex，何時直接輸入名稱就好？

**直接輸入頻道名稱**才是最常用的方式。Regex 只有在頻道名稱有明確命名規律時才值得使用。

**適合用 Regex：**

| 情境 | Regex |
|------|-------|
| 過濾所有含「直播」的頻道 | `直播$` |
| 過濾頻道名含「Game」或「遊戲」 | `(Game\|遊戲)` |
| 過濾政論 / 爆料 / 時事類 | `(政論\|時事\|內幕\|爆料)` |

**不適合用 Regex（直接輸入更快）：**

如果你想同時過濾「話桌點」、「洛狗&阿鶴」、「忽略菌Game」，這三個名稱彼此毫無規律，一條 Regex 無法精準涵蓋，直接各別新增精確名稱即可。

> **原則**：當你發現自己需要用 `(A|B|C)` 列舉完全不相關的名稱，就應改為個別新增精確名稱。

### 常見注意事項

- 特殊字元（`. * + ? ( ) [ ] { } ^ $ | \`）在 Regex 中有特殊意義，若要比對字面字元需加反斜線，如 `\[業配\]`
- 輸入錯誤的 Regex（如括號不匹配）不會報錯，但該規則不會生效
- 可用線上工具（如 [regex101.com](https://regex101.com)）先測試你的 pattern

## 💾 匯出 / 匯入

| 功能 | 說明 |
|------|------|
| **完整設定 JSON** | 備份所有設定（隱藏名單、白名單、關鍵字、過濾設定）；換裝置時匯入還原 |
| **個別 CSV 匯出** | 隱藏名單 / 白名單 / 關鍵字可個別匯出 CSV，用試算表軟體編輯後再匯入 |
| **統計 JSON** | 獨立匯出過濾統計資料（不含在完整設定中） |
| **歷史 CSV** | 獨立匯出隱藏記錄（標題、頻道、原因、時間） |
| **自動備份** | 設定備份間隔（天），自動下載設定 JSON |
| **分享清單** | 產生分享連結，讓他人直接匯入你的過濾設定 |
| **匯入模式** | 「合併」保留現有資料並新增；「覆蓋」完全取代 |

> 統計與歷史記錄**不納入**完整設定備份，需用「📊 統計 JSON」或「📋 歷史 CSV」按鈕獨立匯出。

## ⚙️ 全域設定

在「⚙️ 設定」Tab 底部：

| 選項 | 說明 |
|------|------|
| 🔔 **顯示操作通知（Toast）** | 關閉後，過濾通知不再彈出，但仍寫入面板頂部記錄 |
| 📊 **顯示統計與歷史 Tab** | 關閉後 Tab 隱藏，但後台繼續記錄，仍可在匯出入頁獨立匯出 |

> 透過影片卡片上的快捷按鈕（🚫 / ⭐）觸發的通知，即使關閉 Toast 也**永遠彈出**。

## ❓ 常見問題

**Q：為什麼某些卡片沒有顯示快捷按鈕？**
A：快捷按鈕依賴縮圖載入完成才能正確定位。等待頁面完全載入後，滑鼠重新移過卡片即可。

**Q：語言過濾為什麼沒有生效？**
A：語言偵測基於影片**標題的字元集**。另請確認已點「💾 儲存設定」，儲存後過濾立即生效。

**Q：Regex 輸入後不生效？**
A：常見原因：（1）pattern 格式錯誤（如括號不匹配）；（2）多寫了 `/` 或 `/i`（應直接輸入 pattern）；（3）忘記儲存設定。

**Q：什麼時候該用 Regex，什麼時候直接輸入名稱就好？**
A：頻道名稱有共同命名規律時才值得用 Regex。若要過濾的頻道名稱彼此毫無關聯，直接逐一輸入精確名稱反而更快、更準確。

**Q：統計資料怎麼匯出？**
A：到「💾 匯出入」Tab，使用「📊 統計 JSON」按鈕獨立匯出，不在完整設定備份中。

**Q：用快捷按鈕新增頻道後，面板的清單要切換 Tab 才能看到？**
A：v1.0.4 已修正。快捷按鈕操作後會靜默更新面板清單，下次開啟面板時清單即為最新狀態，不需要切換 Tab。
